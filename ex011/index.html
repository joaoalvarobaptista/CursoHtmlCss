Trabalhando com listas

Listas Ordenadas

São listas que a ordem faz diferença.

A tag < ol > é uma lista que permite que itens sejam incluidos a qualquer momento, mas ela refaz a numeração da lista.

Exemplo:

  1. Acirdar
  2. Ligar para João
  3. Tomar café
  4. Escovar os dentes.

A partir do HTML5, o < /li > tornou-se opcional.

O parametro type, permite que modifiquemos o tipo do indice da lista.

Exemplo com type

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escover os dentes

Outro parametro é o start, que permite dizer ao HTML qual o valor que a lista irá começar

Exemplo:

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escover os dentes

Listas não ordenadas

As listas não ordenadas são aquelas em que a sequência não tem relevância. A tag para iniciar esta lista é < ul >

exemplo:

  • Estudar HTML
  • Estudar Crypto
  • Estudar matemática

Também é possivel mudar o type. Temos os tipos circle, square, disc, todos em minúsculos e disc é o padrão.

Exemplo:

  • Estudar HTML
  • Estudar Crypto
  • Estudar matemática

Juntando Listas

É possível colocar uma lista dentro da outra de forma similar a um aninhamento.

Uma aplicação é a criação de menus

Lista de definições

É uma lista que funciona como um dicionário com termo e definição

Exemplo:
